Vectors

Introduction

Vectors define a direction and magnitude, represented by a line with a directional arrow

Right/up is positive, left/down is negative

Vectors are written as column vectors, with the top being the horizontal movement and bottom vertical


Vector arithmetic

To add vectors, add the components seperately (x+x, y+y), likewise subtracting the same way (x-x, y-y)

When adding vectors the new vector will be from the start of the first to the end of the last vector

Similarly, to multiply vectors multiply the individual components by the desired number or vector.

Multiplying by negative values will flip the vector
